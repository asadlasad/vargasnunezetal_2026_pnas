# vargasnunezetal_2026_pnas
Replication Materials for Reconsideration of Secure Communities Rollout Reveals Preemptive Local-Federal Cooperation in Immigration Enforcement

The Dataverse contains the files and code necessary to replicate the analyses in the manuscript and Supporting Information (SI). To replicate the main results, the only R file needed is VargasNunez_PNAS_MainResults.R. To replicate the SI, the only file needed is VargasNunez_PNAS_Results.R The other two R files show the code that was used to create the panel dataset in the analyses. 

We note that a few of the datasets require users to seek independent access. Interested individuals can contact Syracuse University’s TRAC to file a data usage agreement to secure the detainers, transfers, and removals data. Similarly, we do not provide the Uniform Crime Reporting (UCR) data, but these are available for download via Jacob Kaplan’s Concatenated Files on ICPSR. Lastly, the data on Secure Communities from Ariel White (2016) can be obtained by contacting the author, or via White’s Dataverse.
